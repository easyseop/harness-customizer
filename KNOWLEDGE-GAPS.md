## 충분성 판정 기준

봇이 아래 신호를 감지하면 답변이 **불충분**하다고 판단하고 Gap을 기록한다:

| 신호 | 의미 |
|------|------|
| "일반적으로", "보통은", "아마도" 가 답변에 3회 이상 | 사내 구체 정보 없이 추측 중 |
| 시뮬레이션에서 사내 시스템명 대신 일반 명칭 사용 | 내부 시스템 정보 부재 |
| "담당팀에 확인 필요" 언급 | 봇이 판단 불가 영역 |
| Before/After 차이가 없거나 미미함 | 도메인 맥락 없이 제안 생성 |
